Тип функции `filter`:

```haskell
filter :: (a -> Bool) -> [a] -> [a]
```

Тип функции `map`:

```haskell
map :: (a -> b) -> [a] -> [b]
```

Различия:

`filter` принимает предикат (`a` -> `Bool`) и оставляет только те элементы списка, для которых предикат возвращает `True`. Тип элементов списка не меняется: входной и выходной списки имеют одинаковый тип `[a]`.
map принимает функцию преобразования (`a` -> `b`) и применяет её к каждому элементу, создавая новый список с элементами другого типа `[b]`.
Таким образом, `filter` фильтрует, не меняя тип элементов, а map преобразует каждый элемент, возможно, меняя тип.
